### 🎓 DPLM: Gesture Assistant (Дипломный проект)
**Система распознавания жестов рук в реальном времени с обучением пользователя**

<div align="center">
  <img src="https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white" />
  <img src="https://img.shields.io/badge/OpenCV-27338e?style=for-the-badge&logo=opencv&logoColor=white" />

</div>

- **🎯 Описание:** Инновационная система, которая позволяет пользователю обучать собственные жесты и распознавать их в реальном времени с помощью веб-камеры
- **💡 Ключевые возможности:**
  - Детекция и трекинг рук с помощью MediaPipe
  - Обучение пользовательских жестов с GUI интерфейсом
  - Распознавание в реальном времени с KNN классификатором
  - Озвучивание распознанных жестов (TTS)
  - Поддержка жестов одной и двумя руками
  - Нормализация данных для инвариантности к позиции и масштабу

  <img src="https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white" />

  <img src="https://img.shields.io/badge/Generative%20AI-4285F4?style=for-the-badge&logo=google&logoColor=white" />

- **🎯 Описание:** Проект, продолжающийся с осеннего семестра 2024-2025 учебного года, направленный на создание инновационной системы нейроаватаров в рамках импортозамещения. Система будет способна генерировать речь, видео и предоставлять автоматизированные ответы на голосовые запросы.

- **💡 Проблематика:**
  - Обработка речи
  - Обработка видео и фотографий
  - Работа с ИИ для генерации речи
  - Работа с ИИ для генерации фото и видео изображений
  - Обработка вопросов и генерация ответов

- **🛠️ Текущее состояние:** Ранее использовалось готовое ПО, доступ к которому ограничен. На данный момент имеются нейросети, пригодные для локального развертывания системы.

- **🚀 Ожидаемый результат:** Готовый прототип системы, позволяющий быстро разворачивать новый нейроаватар.

- **🤝 Подразделения ГУАП:** Лаборатория когнитивных исследований и лаборатория робототехники Инженерной школы ГУАП.

- **🗓️ Этапы проекта:**
  1. Изучение классического машинного обучения (на примере прогнозирования стоимости автомобиля)
  2. Изучение технологий, необходимых для создания нейроаватара
  3. Создание итоговой презентации
  4. Выбор лучшей модели для обработки запроса
  5. Генерация ответа
  6. Генерация видео
